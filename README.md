### 👨‍💻 About Me

I am a versatile Software Engineer with a passion for building across the entire stack—from low-level **Embedded Systems** and hardware drivers to modern **Web** and **Mobile Applications**. I love solving complex problems, writing clean code, and creating tools that help people (like my EMDR therapy application, *Vialis*).

- 🔭 I’m currently working on full-stack web applications, embedded drivers, and exploring Go.
- 🌱 I’m constantly learning new frameworks and architectures to expand my toolkit.
- ⚡ Fun fact: I can comfortably context-switch between writing bit-banging C++ code for microcontrollers and spinning up responsive React frontends!
